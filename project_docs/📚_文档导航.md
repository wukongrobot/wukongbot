## 💬 使用场景

### 个人助手
```bash
# 通过飞书与AI助手对话
# AI 可以帮你:
- 回答问题
- 查询信息
- 执行任务
- 管理日程
```

### 团队协作
```bash
# 在企业微信群组中
# @悟空Bot 帮我生成周报
# AI 会自动响应并生成内容
```

### 自动化
```bash
# 定时任务
# 每天早上 9 点发送日报
pnpm wukongbot cron add \
  --schedule "0 9 * * *" \
  --channel feishu \
  --to "chat_id" \
  --message "生成今日工作计划"
```
