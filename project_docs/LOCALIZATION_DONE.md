## 🔧 使用语言包

### 在 Web UI 中使用

```typescript

// 使用翻译
const title = t("overview.title");  // "概览"
const connect = t("common.connect"); // "连接"
```

如需添加新的翻译,编辑 `ui/src/ui/i18n.ts`:

```typescript

  // ... 现有翻译

    title: "新模块",

  },
};
```

## 🎯 汉化原则

### 1. 保持专业性
- 使用准确的技术术语
- 避免过度口语化
- 保持一致的翻译风格

### 2. 符合中文习惯
- 使用中文标点符号
- 保持语序自然
- 适当使用成语和俗语

### 3. 品牌一致性
- 统一使用"悟空Bot"
- 统一使用 🐵 Emoji
- 保持"七十二变"等品牌元素

### 4. 技术准确性
- 保留技术术语原文(如 WebSocket、Token)
- 必要时添加括号注释
- 保持配置项名称一致
