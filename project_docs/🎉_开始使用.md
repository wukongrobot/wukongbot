## 🎯 完整使用流程

### 1. 配置国产大模型

推荐使用 DeepSeek:

```bash
# 创建配置文件
cat > ~/.wukongbot/wukongbot.json <<'EOF'
{
  "models": {
    "providers": {
      "deepseek": {
        "apiKey": "sk-your-key-here",
        "baseUrl": "https://api.deepseek.com",
        "models": [
          {"id": "deepseek-chat", "name": "DeepSeek Chat"}
        ]
      }
    }
  },
  "agent": {
    "model": "deepseek/deepseek-chat"
  }
}
EOF
```

### 2. 配置飞书

```bash
# 运行配置向导
pnpm wukongbot channels onboard feishu

# 或手动配置
cat >> ~/.wukongbot/wukongbot.json <<'EOF'
{
  "channels": {
    "feishu": {
      "appId": "cli_xxxxx",
      "appSecret": "your_secret",
      "webhookPort": 3000,
      "allowFrom": ["*"]
    }
  }
}
EOF
```

### 3. 测试连接

```bash
# 测试飞书连接
pnpm wukongbot channels status feishu --probe

# 发送测试消息
pnpm wukongbot message send \
  --channel feishu \
  --to "chat_id" \
  --message "你好，悟空!"
```

### 4. 启动 Gateway

```bash
# 启动 Gateway
pnpm wukongbot gateway --port 18789 --verbose
```

### 5. 打开 Web 管理后台

```bash
# 在新终端
cd /root/code/wukongbot/ui
pnpm install
pnpm dev

# 访问 http://localhost:5173
```

你会看到**悟空Bot 控制面板**，所有界面都是中文！

## 🔍 常见问题

### Q: 构建失败怎么办？
A: 确保已经运行 `pnpm install`，Node.js 版本 >= 22

### Q: 配置向导是英文的？
A: 需要先构建项目: `pnpm build`

### Q: Web UI 没有汉化？
A: 清除浏览器缓存后刷新

### Q: 如何升级 Node.js？
A:
```bash
nvm install 22
nvm use 22
```

### Q: 飞书连接失败？
A: 检查：
1. App ID 和 Secret 是否正确
2. 应用是否已发布并启用
3. Webhook URL 是否配置正确
