## 🎯 接下来做什么？

### 1. 构建项目

```bash
cd /root/code/wukongbot
pnpm build
```

这将编译 TypeScript 代码到 `dist/` 目录。

### 2. 运行配置向导

```bash
# 运行悟空Bot配置向导
pnpm wukongbot onboard
```

向导会引导你完成：
- ✅ 选择配置模式（快速开始 / 手动配置）
- ✅ 配置 AI 模型（DeepSeek、千问等）
- ✅ 配置频道（飞书、企微、钉钉等）
- ✅ 设置工作空间

### 3. 查看 CLI 效果

```bash
# 查看帮助（应该显示🐵悟空Bot Banner）
pnpm wukongbot --help

# 查看频道列表
pnpm wukongbot channels list

# 查看版本
pnpm wukongbot --version
```

### 4. 配置国产 IM 平台

#### 飞书
```bash
pnpm wukongbot channels onboard feishu
```

#### 企业微信
```bash
pnpm wukongbot channels onboard wecom
```

#### 钉钉
```bash
pnpm wukongbot channels onboard dingtalk
```

### 5. 启动 Web 管理后台

```bash
# 启动 Gateway
pnpm wukongbot gateway --port 18789

# 在另一个终端启动 Web UI (开发模式)
cd ui
pnpm install
pnpm dev

# 访问 http://localhost:5173
# 应该看到 "悟空Bot 控制面板"
```

## ⚠️ 已知问题

### Node.js 版本警告
```
当前版本: v21.5.0
要求版本: >= 22.12.0
```

虽然可以运行，但建议升级：
```bash
nvm install 22
nvm use 22
```

### node-llama-cpp 构建失败
```
错误: cmake not found
```

这是可选依赖，不影响核心功能。如需使用本地模型，需要安装 cmake:
```bash
# Ubuntu/Debian
sudo apt-get install cmake

# macOS
brew install cmake
```
