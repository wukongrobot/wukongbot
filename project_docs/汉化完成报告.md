1. **保持专业性**
   - 使用准确的技术术语
   - 保持专业的表达方式

2. **符合中文习惯**
   - 使用中文标点符号
   - 语序自然流畅

3. **品牌一致性**
   - 统一使用 "悟空Bot" 和 🐵
   - 保持孙悟空主题元素

4. **代码兼容性**
   - 保留代码内部的英文变量名
